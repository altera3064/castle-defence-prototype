# Castle Defence Prototype

Browser-based castle defense prototype.

## Run Locally

Open `index.html` in a browser, or run the local server shortcut script:

```powershell
powershell -ExecutionPolicy Bypass -File .\start_game.ps1
```

The game runs at:

```text
http://127.0.0.1:5173/index.html
```

## Current Prototype

- Human side on the left, monster side on the right
- Buildable walls, gates, arrow towers, barracks, and catapult workshop
- Melee, archer, and catapult squads
- Monster waves with scaling enemies and boss waves
- Canvas-based controls and battle simulation
